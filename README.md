# 🥋 Street Fighter JS – Proyecto Arcade en JavaScript

Un juego tipo **Street Fighter** desarrollado con **JavaScript**, **HTML5 Canvas** y **CSS**, donde dos personajes luchan en escenarios inspirados en **localizaciones de España**.  
Este proyecto forma parte del aprendizaje de desarrollo web y programación orientada a objetos.

---

## 🎮 Características principales

- Motor de juego creado desde cero con JavaScript.
- Sistema de **sprites animados** para cada personaje.
- Detección de colisiones y sistema de daño.
- Fondos estilo arcade basados en lugares icónicos de España.
- Pantalla de **selección de personaje**.
- Pantalla de **Game Over** y reinicio.
- Controles independientes para Player 1 y Player 2.
- Diseño responsive para adaptarse a diferentes pantallas.

---

## 🕹️ Controles

### **Player 1**
- **←** → Mover izquierda  
- **→** → Mover derecha  
- **↑** → Saltar  
- **1** → Atacar
- **0** → Atacar
  
### **Player 2**
- **A** → Mover izquierda  
- **D** → Mover derecha  
- **W** → Saltar  
- **Q** → Atacar
- **E** → Atacar
